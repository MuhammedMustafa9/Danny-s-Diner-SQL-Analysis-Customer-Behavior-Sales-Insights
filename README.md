# 🍣 Danny’s Diner SQL Analysis

## 🧠 Overview
This project analyzes restaurant sales data using SQL to uncover customer behavior, spending patterns, and product performance.

## 🎯 Objectives
- Analyze customer visits and spending
- Identify popular menu items
- Evaluate customer behavior before and after membership
- Analyze loyalty program performance

## 📊 Key Analysis
- Total spending per customer
- Visit frequency
- Most popular items
- First purchase before and after membership
- Points and loyalty analysis

## 🚀 Features
- SQL joins across multiple tables
- Aggregations and grouping
- Window functions (ranking)
- Business-driven analysis

## 🛠 Tools Used
- SQL
- Data Analysis
